# Discord Integration

Discord support is useful when customers ask questions in a community or
developer server. Woes keeps those conversations visible to operators and lets
teams apply the same triage, issue, and analytics model used for other channels.

## What Discord Adds

- Developer-community questions in Woes Inbox.
- Shared triage with live chat and email.
- Operator visibility into recurring topics.
- Issue creation for Discord-reported bugs or requests.

## Setup Path

1. Review [Discord](/channels/discord).
2. Connect the workspace Discord integration.
3. Choose the channels Woes should observe.
4. Send a test message and confirm it appears in Inbox.
5. Apply tags or create issues for durable work.

Production Discord intake requires the gateway worker unless you have deployed
another server-side Discord event source. For bot reinvite recovery and delivery
diagnostics, see [Discord](/channels/discord) and
[Troubleshooting](/reference/troubleshooting).
