# HubSpot Developer Notes

For the operational workflow, commands, tables, and troubleshooting checklist, start with [HubSpot](/integrations/hubspot).

## Main Components

| Component | Purpose |
| --- | --- |
| `IqSync/ApiWrapper/HubSpot.cs` | Outbound CRM sync for companies, contacts, deals, deal updates, line items, and gross margin. |
| `IqConnectWebhooksAsync` | Azure Durable Functions flow for HubSpot quote acceptance and IQ SO creation. |
| `integration_settings` | Per-tenant HubSpot configuration. |
| `hubspotdealstage` | Tenant-specific deal stage mapping. |

## Developer Checks

| Area | What To Check |
| --- | --- |
| Authentication | Confirm whether the tenant uses bearer-token mode or API-key mode. |
| Deal stages | Validate `hubspotdealstage` mappings before debugging missing or incorrect deal updates. |
| Duplicate contacts | Check `iqconnect_crm_duplicate_contacts` when a contact does not sync. |
| Gross margin | Check the gross-margin query and sync flags before re-running `sogm`. |
