# Feedapi

## IqFeedApi — Feed Data REST API 

ASP.NET Web API 2 project that provides programmatic REST access to IQConnect feed data. Includes auto-generated API documentation via the HelpPage area. 

### Structure 

 | Component | Purpose 
| --- | --- |
 | Controllers/HomeController.cs | Landing page 
 | Controllers/ValuesController.cs | API endpoints for feed data access 
 | Areas/HelpPage/ | Auto-generated API documentation (XML doc-based)
