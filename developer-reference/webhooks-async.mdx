# Webhooks Async

## IqConnectWebhooksAsync — Durable Functions Orchestrator 

Azure Durable Functions project that handles long-running, multi-step webhook workflows. Primary use case: converting a HubSpot quote acceptance into an IQ Reseller sales order with full line item resolution and master item creation.

### Orchestration Flow 

The CrmIntegrationAync.RunOrchestrator() method dispatches based on action type: 

 | Action | Activity Function | Description 
| --- | --- | --- |
 | heartbeat | CheckHeartbeat (x3) | Runs 3 sequential heartbeat checks to verify orchestrator health 
 | (default) | CreateSO | Full quote-to-SO conversion workflow 

### CreateSO Workflow 
Parse webhook payload 

Identify tenant (iq_tenant) 

Load API keys from env vars 

Login to IQ API 

Fetch full quote from HubSpot API 

Resolve line items + master items 

Create SO via IQ API 

Return result 

The autoAddMissingItem flag (from the webhook payload) controls whether unrecognized items are auto-created as master items in IQ Reseller during SO creation.
