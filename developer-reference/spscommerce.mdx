# SPS Commerce Developer Notes

For the operational workflow, commands, recent changes, and troubleshooting checklist, start with [SPS Commerce](/integrations/sps-commerce).

## Main Components

| Component | Purpose |
| --- | --- |
| `SpsCommerce.cs` | Main wrapper for feed preparation, XML generation, and SFTP transfer. |
| `SpsCommerceDS.cs` | Data structures for SPS document types. |
| `SpsFormatConverter.cs` | Date, address, and UOM conversion helpers. |
| `from-sps solocal` | Local replay path for testing inbound SPS payloads. |

## Developer Checks

| Area | What To Check |
| --- | --- |
| Address errors | Review CR 125 limits and the generated invoice XML. |
| Line mismatch | Compare SPS PO line number, IQ SO line number, and internal comments. |
| ASN errors | Check whether CR 126A logic applies to the failing shipment. |
