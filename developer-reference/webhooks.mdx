# Webhooks

## IqConnectWebhooks — HTTP Azure Functions 

Azure Functions project (isolated worker model, .NET) that exposes HTTP endpoints for inbound webhooks. Lightweight entry point that validates requests and forwards payloads to the async orchestrator. 

### Functions 

 | Function Name | Trigger | Purpose 
| --- | --- | --- |
 | HeartBeat | HTTP GET/POST (Anonymous) | Health check endpoint. Returns timestamp + "alive and well" message. 
 | PostQuote | HTTP GET/POST (Anonymous) | Receives HubSpot quote webhook payload. Reads request body, logs it, returns OK. (Forwarding to async orchestrator is configured externally.)
