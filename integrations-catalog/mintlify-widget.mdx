# Mintlify Widget Integration

The Mintlify widget integration places Woes live chat directly inside a Mintlify documentation site.

## Status

**Available now.** Woes has documentation for installing the widget in Mintlify.

## What it does

The widget gives developers a support entry point while they are reading API documentation:

- Visitors can start live chat from the docs site.
- Widget traffic routes into the Woes workspace and inbox.
- The selected Woes agent can answer from workspace-scoped API context.
- Operators can take over when the agent is not confident or when a human response is needed.

## Where it appears in Woes

- **Widget settings** configure visual behavior, identity, routing, launch behavior, and install snippets.
- **Mintlify docs** host the widget script or embed snippet.
- **Inbox** receives conversations created from documentation visitors.

## Recommended setup

- Use an agent whose context matches the docs site being embedded.
- Configure widget identity verification before sending authenticated customer details.
- Test on staging docs before enabling on production documentation.
- Use source coverage analytics to improve answers for repeated docs questions.
